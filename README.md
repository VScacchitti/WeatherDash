# WeatherDash
WeatherDash is your weather when you need it in a flash! WeatheDash allows you to seach for current and 5-day extended forcast conditions for any city just by searching! This app uses an open weather API to pull current conditions such as temperature, humidity and wind speed!

<img width="1853" alt="Screen Shot 2020-09-11 at 2 29 23 PM" src="https://user-images.githubusercontent.com/67161794/92960945-736d9d00-f43c-11ea-9d4b-1ec87cbe9abb.png">

When you search for a new city it as added to the top of the list as the most recent searched city. Forcast for any saved search can be brought up by clicking on it.

<img width="1538" alt="Screen Shot 2020-09-11 at 2 34 44 PM" src="https://user-images.githubusercontent.com/67161794/92961093-b0d22a80-f43c-11ea-8743-4a9f5908e56d.png">

Searches are stored locally, so upon refresh previous searches will be available. 

<img width="1542" alt="Screen Shot 2020-09-11 at 2 35 16 PM" src="https://user-images.githubusercontent.com/67161794/92961168-d2331680-f43c-11ea-9932-c50a191d99d4.png">

This application uses HTML, CSS, JavaScript, JQuery, and OpenWeatherAPI!
